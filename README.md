# سِراج — المستشار المالي الذكي 🌟

**سِراج** هو مساعد مالي ذكي مبني للمملكة العربية السعودية، يساعد الأفراد على تتبع إنفاقهم وإدارة ميزانياتهم وتحقيق أهدافهم المالية — بلهجة سعودية ودية ووعي تام بأحكام الشريعة الإسلامية.

> 🏆 مقدَّم ضمن مسابقة هاكاثون — النسخة النهائية

---

## 📋 نظرة عامة

| العنصر | التفاصيل |
|--------|----------|
| 🌐 نوع التطبيق | تطبيق ويب (Fullstack) |
| ⚙️ الباكيند | FastAPI (Python 3.11+) |
| 🤖 الذكاء الاصطناعي | GPT عبر Replit AI Integration |
| 🗄️ قاعدة البيانات | SQLite + SQLAlchemy Async |
| 🔐 المصادقة | JWT Tokens |

---

## ✨ المميزات الرئيسية

- **🤖 سِراج AI** — شات بوت ذكي يقرأ بياناتك المالية الحقيقية ويجيب بالعربية السعودية
- **📊 لوحة التحكم** — نظرة شاملة على الدخل والإنفاق ومؤشر الصحة المالية
- **💰 تتبع المعاملات** — تسجيل وتصنيف كل معاملة مالية
- **📋 الميزانيات** — تحديد حدود الإنفاق لكل فئة ومتابعتها
- **🏦 الحصالات** — خطط ادخار ذكية لأهداف محددة
- **🎯 الأهداف المالية** — تتبع الأهداف طويلة المدى
- **📈 الاستثمارات** — تسجيل ومتابعة المحفظة الاستثمارية
- **🏠 التمويل الإسلامي** — دعم كامل لحسابات التمويل المتوافق مع الشريعة
- **🚨 التنبيهات الذكية** — إشعارات عند الاقتراب من حدود الميزانية

## 🚀 تشغيل المشروع محلياً

### المتطلبات الأساسية
- Python 3.11+
- pip

### خطوات التشغيل

```bash
# 1. استنساخ المشروع
git clone https://github.com/<your-username>/siraj-final.git
cd siraj-final

# 2. إنشاء بيئة افتراضية
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. تثبيت المتطلبات
pip install -r requirements.txt

# 4. إعداد متغيرات البيئة
cp .env.example .env
# عدّل .env وأضف مفاتيح API المطلوبة

# 5. تشغيل الخادم
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## 📡 نقاط API الرئيسية

| المسار | الوصف |
|--------|-------|
| `POST /api/v1/auth/register` | تسجيل مستخدم جديد |
| `POST /api/v1/auth/login` | تسجيل الدخول والحصول على Token |
| `GET /api/v1/dashboard/overview` | نظرة عامة مالية |
| `GET /api/v1/dashboard/health-score` | مؤشر الصحة المالية |
| `GET /api/v1/transactions/` | قائمة المعاملات |
| `POST /api/v1/transactions/` | إضافة معاملة جديدة |
| `GET /api/v1/budgets/` | الميزانيات |
| `GET /api/v1/savings/` | الحصالات |
| `POST /api/v1/chat/` | الدردشة مع سِراج AI |

> 📄 للاطلاع على التوثيق الكامل لجميع نقاط API: [`api_endpoints.md`](./api_endpoints.md)

## 🏗️ هيكل المشروع

```
siraj-final/
├── backend/
│   ├── app/
│   │   ├── ai/                  # محرك الذكاء الاصطناعي
│   │   │   ├── agent_loop.py    # حلقة معالجة رسائل المستخدم
│   │   │   ├── system_prompt.py # شخصية سِراج وتعليماته
│   │   │   ├── context_builder.py # بناء السياق المالي للمستخدم
│   │   │   ├── ai_provider.py   # مزود الذكاء الاصطناعي
│   │   │   └── tools.py         # أدوات الـ AI (قراءة/كتابة البيانات)
│   │   ├── models/              # نماذج قاعدة البيانات (SQLAlchemy)
│   │   ├── routers/             # مسارات API (FastAPI)
│   │   ├── schemas/             # مخططات البيانات (Pydantic)
│   │   ├── config.py            # إعدادات التطبيق
│   │   ├── database.py          # إعداد قاعدة البيانات
│   │   └── main.py              # نقطة الدخول الرئيسية
│   └── .env.example             # مثال على متغيرات البيئة
├── api_endpoints.md             # توثيق شامل لنقاط API
├── Siraj_API_Postman_Collection.json  # مجموعة Postman جاهزة
└── README.md                    # هذا الملف
```

## 🧠 كيف يعمل سِراج AI

```
رسالة المستخدم
       ↓
  agent_loop.py
       ↓
context_builder.py  →  بيانات مالية حقيقية من قاعدة البيانات
       ↓
system_prompt  →  شخصية سِراج + السياق المالي المخصص
       ↓
GPT (عبر Replit AI Integration)
       ↓
Streaming SSE  →  الرد كلمة بكلمة للواجهة
```
